# Assessment Model

Each assessment is a structured object:

Assessment
- Metadata
- Context (persona, workload)
- Inputs (customer + platform)
- Evidence (probes)
- Findings
- Scores
- Report

## Lifecycle
1. Create assessment
2. Populate platform data
3. Collect customer context
4. Run probes
5. Generate findings
6. Score
7. Review
8. Deliver report
