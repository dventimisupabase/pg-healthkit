# PostgreSQL Health Assessment Framework

This repository contains a full-stack methodology and implementation scaffold
for evaluating PostgreSQL database health in a managed service context (e.g. Supabase).

It includes:
- Conceptual framework (personas, objectives, workload classification)
- Assessment model and lifecycle
- Probe catalog and SQL queries
- Rule engine design
- Data model and CLI contract
- Future roadmap

This pack is intended to bootstrap agentic development (e.g. Codex CLI).
